# FUTURE_FS_01 - Task 1: My Personal Portfolio Website

## Project Overview
A full stack personal portfolio website built with React, Node.js, and MongoDB. Designed with a dark techy aesthetic, purple accents, and interactive animations.


## Features
- Responsive design across all screen sizes
- Dark / Light mode toggle
- Typing animation on homepage
- Animated skill bars
- Masonry project grid with filter buttons
- Contact form with MongoDB storage and email notifications
- React Router for multi-page navigation
- Floating photo frame with animations


## 🛠️ Tech Stack

**Frontend**
- React.js
- CSS3 (custom, no frameworks)
- React Router DOM

**Backend**
- Node.js
- Express.js
- Nodemailer

**Database**
- MongoDB Atlas



## Setup & Installation
To get started with this project, follow these steps:

### Prerequisites
- Node.js installed
- MongoDB Atlas account
- Gmail account with App Password

### 1. Clone the repository
```bash
git clone https://github.com/NadaK110/FUTURE_FS_01.git
cd FUTURE_FS_01
```

### 2. Setup the Frontend
```bash
cd my-portfolio
npm install
npm start
```

### 3. Setup the Backend
```bash
cd ../backend
npm install
```
Then run:
```bash
node server.js
```

### 4. Open in browser

http://localhost:3000
